# E-Commerce Product Page

A modern and responsive e-commerce product page built using HTML, CSS, and JavaScript.

## 🚀 Features
- Premium sneaker product UI
- Product image gallery
- Thumbnail image switching
- Size selection section
- Quantity increase/decrease
- Add to Cart and Buy Now buttons
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
